Image Search Abstraction Layer

User Stories

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

To execute a search :

https://dlukimagesearch.herokuapp.com/api/imagesearch/bears

To paginate results add ?offset= followed by the number you would like to paginate by :

https://dlukimagesearch.herokuapp.com/api/imagesearch/bears?offset=10

You can view the latest 10 searchs by executing the following :

https://dlukimagesearch.herokuapp.com/api/latest
